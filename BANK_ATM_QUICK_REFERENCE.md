## ✏️ EDIT HEADING (3 Easy Steps)

2. Go to **Content Management** → **Manage Pages** → **bank-atm**
3. Edit **hero-section** block:
   - Change `title` field (currently: "Bank/ATM")
   - Change `subtitle` field (currently: "Banking Services on Campus")
   - Change `description` field
4. **Save** ✅

## 📸 CHANGE IMAGES (2 Easy Methods)

### **Method 1: Replace Files Directly**
1. Go to folder: `Frontend/public/images/facilities/`
2. Replace `atm1.jpg` and `atm2.jpg` with your photos
3. Keep same filenames → Changes appear automatically! ✅

### **Method 2: Upload via Admin Panel**
1. Login to Admin Panel
2. Go to **Content Management** → **Manage Pages** → **bank-atm**
3. Find **atm-image-1** or **atm-image-2** blocks
4. Click **Edit**
5. Upload new image or change URL path
6. Update caption text if needed
7. **Save** ✅
